HIGWAY FATALITIES IN THE U.S.

    Methodology

    The viz leverages dc.leaflet.js javascript library created by Boyan Yurukov.

    Under the hood, the library uses leaflet.js and dc.js(built up on crossfilter and d3 libraries).

    For more information, please visit http://dc-js.github.io/dc.leaflet.js.

    Data

    The original data can be accessed from the National Highway Traffic Safety Adminstration.

    The data used in this visualization contains the location, date and time of fatalities.

    You can download the data here.

    About

    The visualization shows motor vehicle related fatalities in the U.S. in the year 2015.

    The base for the cluster map contains live traffic data.

    To interact with the visualization, zoom-in or select features from the graphs.
